Free blogging with Hugo and GitHub Pages

Quick guide to your own static blog hosted in GitHub and served on your own domain.

Hugo

Quick start to your static site with Hugo:

1. Install Hugo

brew install hugo
# or
port install hugo

2. Create new blog

hugo new site hugo-blog

2.1 Customize your blog

Edit the config.toml file change the title as you wish if you intend to use a custom domain you may want to update the baseURL as well. To be able to publish via GitHub Pages add the following line to the file:

publishdir = "docs"

3. New post

cd hugo-blog
hugo new posts/my-first-post.md

Modify the file posts/my-first-post.md to write your first post. To publish the post change the draft: to false. You may also add tags and categories to the post like:

categories: [test, publishing]
tags: [hugo, content, static site generator]

4. Add to git

Create your git repository hugo-blog

git init
git remote add origin git@github.com:[your-git-user]/hugo-blog.git

5. Add git pre-commit hook

To build the pages Rename the file pre-commit.sample in pre-commit (location: hugo-blog/.git/hooks) and replace the contents of the file with:

#!/bin/sh
#
# Run Hugo build and add the generated files to the commit
#
echo "Pre-commit hook start..."

echo "Building Hugo distribution"
hugo

echo "Adding new generated files to commit"
git add docs/*

echo "Pre-commit hook done."

6. Test

To test how your site looks run hugo server -D (-D enables drafts to be shown)

7. Commit your changes
